◆ neorv32_cpu_sleep()

void neorv32_cpu_sleep ( void  )
inline

Put CPU into "sleep" mode.

Note
This function executes the WFI insstruction. The WFI (wait for interrupt) instruction will make the CPU stall until an interupt request is detected. Interrupts have to be globally enabled and at least one external source must be enabled (e.g., the CLIC or the machine timer) to allow the CPU to wake up again. If 'Zicsr' CPU extension is disabled, this will permanently stall the CPU.
